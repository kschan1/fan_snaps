Fan-Snaps is a photo sharing app for sports fan made using Ruby on Rails.

[Demo Link](https://fan-snaps.herokuapp.com)

### Goal:
More Rails practice.

### Technologies:
HTML, CSS, Ruby on Rails, PostgreSQL, Bootstrap, CarrierWave, Amazon S3, Fog.
